#Project 1: Real-time Embedded System for Home Automation



Description:

In this project, I led a team of members to design and develop a real-time embedded system that provides advanced home automation and security features. The system includes a central control unit connected to various sensors, actuators, and cameras. Users can remotely monitor and control their home environment through a mobile app or a web interface. The project involved hardware design, firmware development, and mobile app programming. We implemented features such as remote door locking, temperature control, and intruder detection, making it a comprehensive smart home solution.


My Role:

  1. Project Lead
  2. Hardware Design
  3. Firmware Development
  4. Mobile App Development
  5. System Integration


Technologies and Tools:

  • Microcontroller Programming (Arduino)
  • IoT Protocols (MQTT)
  • Mobile App Development (Android and iOS)
  • PCB Design (Eagle)
  • Web Development (HTML, CSS, JavaScript)


#Project 2: High-Performance GPU Accelerated Image Processing





Description:

As a part of my senior thesis, I worked on optimizing image processing algorithms for high-performance computing using GPU acceleration. We aimed to reduce the time required for complex image analysis tasks in medical imaging and computer vision applications. I implemented parallel programming techniques using CUDA and OpenCL to harness the power of GPUs. The result was a significant reduction in processing time, enabling faster and more accurate image analysis for medical diagnoses and object recognition.


My Role:

  1. Research and Development
  2. Parallel Algorithm Optimization
  3. GPU Programming (CUDA and OpenCL)
  4. Performance Benchmarking
  5. System Integration


Technologies and Tools:

  • NVIDIA CUDA
  • OpenCL
  • C/C++ Programming
  • MATLAB
  • Performance Profiling (NVIDIA Nsight, AMD CodeXL)
